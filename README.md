# 🚀 Alokit Mishra Portfolio

A modern and interactive 3D portfolio website built using React, TypeScript, Three.js, GSAP, and WebGL.

This portfolio showcases my projects, achievements, experience, and work in Artificial Intelligence, Machine Learning, and Software Development.

## ✨ Features

* Interactive 3D experience with Three.js
* Smooth animations using GSAP
* Responsive design for desktop and mobile
* Modern React + TypeScript architecture
* Project showcase section
* Resume download support
* Contact and social links integration

## 🧰 Tech Stack

* React
* TypeScript
* Three.js
* GSAP

## 🚀 Run Locally

### Clone Repository

```bash
git clone https://github.com/alokitm/alokit-portfolio.git
cd alokit-portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
